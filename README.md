# Rnes

[![CircleCI](https://circleci.com/gh/r7kamura/rnes.svg?style=svg)](https://circleci.com/gh/r7kamura/rnes)

A NES emulator written in Ruby.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'rnes'
```

And then execute:

```sh
bundle
```

Or install it yourself as:

```sh
gem install rnes
```
